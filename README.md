# Habit Tracking Analytics

Sistema de análisis de hábitos diarios orientado a medir cumplimiento, consistencia y evolución en el tiempo mediante métricas claras y visualizaciones simples.  
El proyecto está diseñado con un enfoque analítico, priorizando la toma de decisiones por sobre la estética.

Permite registrar hábitos diarios y obtener insights sobre desempeño general, hábitos críticos y tendencia temporal.

## Funcionalidades principales

- Registro diario automatizado de hábitos
- Cálculo de métricas de cumplimiento
- Identificación de hábitos críticos (cuellos de botella)
- Análisis de evolución mensual
- Visualización clara y orientada a decisiones
- Automatización mediante Google Apps Script

---

## Cómo usar el proyecto

1. Abrir el enlace indicado en el archivo `link.txt`
2. Ir a **Archivo → Crear una copia**
3. Seleccionar la ubicación donde se guardará la hoja  
   *(El script se copiará automáticamente)*
4. Definir los hábitos en la hoja **Interfaz Diaria**
5. Al finalizar el día, presionar el botón **Registrar Día**
6. En el primer uso, otorgar permisos a Google Apps Script cuando se solicite

---

## Preguntas que responde el análisis

- ¿Cumplo o no cumplo mis hábitos?
- ¿Estoy mejorando o empeorando con el tiempo?
- ¿Qué hábitos funcionan bien y cuáles son el principal cuello de botella?

---

## Métricas principales

- **% de cumplimiento general**
- **Cantidad de días cumplidos**
- **Promedio de cumplimiento por hábito**
- **Tendencia mensual** (sube / baja / estable)

---

## Enfoque del proyecto

Este proyecto prioriza:
- claridad analítica
- métricas accionables
- simplicidad estructural
- aprendizaje práctico de análisis de datos

No busca ser un dashboard corporativo, sino una herramienta funcional para entender y mejorar el comportamiento a partir de datos reales.
